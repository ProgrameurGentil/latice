# SAE

# Présentation du sujet
Nous devons refaire le jeu de plateau latice.
Nous avons donc refait le jeu en Java/JavaFX.

# Comment y jouer ?
Dans le depot, aller dans "src", "main", "java/latice", "application". Dans ce dossier se trouve un fichier "laticeMain.java". C'est le fichier à executer pour pouvoir joueur.

# Comment jouer ?
Au tour 1, le premier joueur à jouer vien placer une tuile sur la case lune, signifiant ainsi le début de la partie, ensuite le joueur devra appuyer sur le bouton du menu pour passer son tour.
Le joueur suivant viendra poser une tuile de la même couleur ou de la même forme sur un coté adjacent à la tuile, si la tuile ne peut pas être posée, alors la case se colorera en rouge.
Au fil de la partie, les joueur accumuleront des points, soit en posant une tuile sur une case soleil, soit en faisant l'une de ses combinaisont:
-Double: un double est quand le joueur vient poser une tuile adjacemment à 2 tuiles en même temps
-Triple: un triple est quand le joueur vient poser une tuile adjacemment à 3 tuiles en même temps
-Latice: un latice est quand le joueur vient poser une tuile adjacemment à 4 tuiles en même temps (il se retrouvera donc entouré de 4 autres tuiles)

Avec ses points, un joueur pourra faire 2 actions:
-mettre sa main dans la pool et piocher 5 autres tuiles
-jouer un autre coup

Bon jeu.
